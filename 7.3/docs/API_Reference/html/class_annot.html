Annot::setBorderColor (   borderColor)
inline

Set annotation's border color.

Parameters
borderColor{*} - Specify border color. Supported color format:
  • number, such as 0xff0000, 16711680
  • PDF color array, such as ['RGB',1,1,1] which means white.
  • HTML color format, such as '#ff0000', 'rgb(255,0,0)', 'rgba(255,0,0,1)'
Returns
{Promise.<T>|*} - A boolean value indicated whether success.

◆ setBorderInfo()

Annot::setBorderInfo (   borderInfo)
inline

Set annotation's border information.

Note
The cloudy segments are only honored by polygon, square and circle. If the cloud style was applied to others like text markup, caret, stamp, ink and note, no effect will be drawn.
Parameters
borderInfo{object} - Border infomation.
[borderInfo.cloudIntensity]{Integer} - Cloud intensity
[borderInfo.dashPhase]{number} - Dash phase.
[borderInfo.dashes]{Array<number>} - Dashes array.
borderInfo.style{number} - Border style, refer to Border_Style.
borderInfo.width{number} - Border width.
{
cloudIntensity:{number},//Intensity of the cloudy effect, only 0,1 and 2 is valid.
dashPhase:{number},//Dash phase.
dashes:Array[number],//A dash array that represents the dash patterns.
style:{number},
width:{number}, // unsigned integer.
}
Returns
{Promise.<T>}
