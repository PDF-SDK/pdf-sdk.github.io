Detailed Description

Scroll bar container component
Controls the scrolling of a pdfviewer view
.

examples:
function example () {
new PDFViewer({
customs: {
ScrollWrap: class CustomScrollWrap extends ScrollWrap {
constructor (viewerRender) {
super(viewerRender)
}
}
}
})
}
