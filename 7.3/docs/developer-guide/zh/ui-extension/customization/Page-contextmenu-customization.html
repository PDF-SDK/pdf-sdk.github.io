# 自定义页面右键菜单

本文将帮助您了解，页面级右键菜单组件名称列表，如何通过fragments和template来删除内置页面右键菜单项，新增或插入自定义菜单项，以及如何通过fragmets及viewerUI来隐藏内置页面右键菜单项。

# 页面右键菜单组件名称为

fv--page-contextmenu, 内有这些子菜单项元素，按照显示的顺序如下：

  • fv--contextmenu-item-full-screen
  • fv--conextmenu-item-select-tool
  • fv--conextmenu-item-hand-tool
  • fv--contextmenu-item-marquee-zoom
  • fv--conextmenu-item-zoom-actual-size
  • fv--conextmenu-item-zoom-fitpage
  • fv--conextmenu-item-zoom-fitwidth
  • fv--contextmenu-item-print
  • fv--conextmenu-item-search
  • fv--file-property-contextmenu

# 如何删除其中的一项

下面例子代码执行后，会删除目标右键菜单项Actual Size。

new PDFUI({
  // the other options ...
  fragments: [
    {
      target: "fv--conextmenu-item-zoom-actual-size",
      action: UIExtension.UIConsts.FRAGMENT_ACTION.REMOVE
    }
  ]
});

# 如何替换其中一项

下面例子代码执行后，自定义菜单元素customize contextmenu item 会替换Actual Size。点击新元素后执行callback中定义的操作。

new PDFUI({
  // the other options ...
  fragments: [
    {
      target: "fv--conextmenu-item-zoom-actual-size",
      action: UIExtension.UIConsts.FRAGMENT_ACTION.REPLACE,
      template: `<contextmenu-item name="custom-contextmenu-item">customize contextmenu item</contextmenu-item>`,
      config: [
        {
          target: "custom-contextmenu-item",
          callback: function() {
            alert("contextmenu item clicked!");
          }
        }
      ]
    }
  ]
});

# 如何在其中的一项后面插入其它菜单项

下面例子代码执行后，会在Actual Size后新增一项自定义菜单。点击新菜单后执行callback中定义的操作。如果要在前面插入，则把action 改成 UIExtension.UIConsts.FRAGMENT_ACTION.BEFORE

new PDFUI({
  // the other options ...
  fragments: [
    {
      target: "fv--conextmenu-item-zoom-actual-size",
      action: UIExtension.UIConsts.FRAGMENT_ACTION.AFTER,
      template: `<contextmenu-item name="custom-contextmenu-item">customize contextmenu item</contextmenu-item>`,
      config: [
        {
          target: "custom-contextmenu-item",
          callback: function() {
            alert("contextmenu item clicked!");
          }
        }
      ]
    }
  ]
});

# 如何隐藏页面右键菜单

  1. 第一种做法是通过fragments的配置项加class样式，强行隐藏
new PDFUI({
  // the other options ...
  fragments: [
    {
      target: "fv--page-conextmenu",
      config: {
        cls: "fv__ui-force-hide"
      }
    }
  ]
});

这种做法的效果就是右键点击后没有任何响应。

  1. 第二种方法是自定义 viewerUI
new PDFUI({
  viewerOptions: {
    viewerUI: new (class extends UIExtension.XViewerUI {
      createContextMenu(owner, anchor, config) {
        switch (owner) {
          case PDFViewCtrl.STATE_HANDLER_NAMES.STATE_HANDLER_HAND:
          case PDFViewCtrl.STATE_HANDLER_NAMES
            .STATE_HANDLER_SELECT_TEXT_ANNOTATION:
            return;
        }
        return super.createContextMenu(owner, anchor, config);
      }
    })()
  }
});

这种做法，页面会不显示内置的右键菜单， 但显示浏览器自带的右键菜单。

# 如何显示自定义的右键菜单组件

显示自定义右键菜单组件，需要通过覆盖 viewerUI 的实现：

new PDFUI({
  viewerOptions: {
    viewerUI: new (class extends UIExtension.XViewerUI {
      createContextMenu(owner, anchor, config) {
        switch (owner) {
          case PDFViewCtrl.STATE_HANDLER_NAMES.STATE_HANDLER_HAND:
          case PDFViewCtrl.STATE_HANDLER_NAMES
            .STATE_HANDLER_SELECT_TEXT_ANNOTATION:
            return new (class extends PDFViewCtrl.IContextMenu {
              constructor() {
                super();
                this.initContextmenu();
              }
              destroy() {
                $(anchor).contextMenu("destroy");
              }
              showAt(x, y) {
                $(anchor).contextMenu();
              }
              disable() {
                super.disable();
                $(anchor).contextMenu("destroy");
              }
              enable() {
                super.enable();
                this.initContextmenu();
              }
              initContextmenu() {
                // 下列以Jquery contextmenu 插件为例。需按顺序引用 contextMenu.min.css, jquery.min.js 和 contextMenu.min.js 才能正确运行示例。
                $(anchor).contextMenu({
                  selector: config.selector,
                  items: [
                    {
                      name: 'show "Hello World"',
                      callback: function() {
                        alert("hello world");
                      }
                    },
                    {
                      name: 'show "Bye!"',
                      callback: function() {
                        alert("Bye!");
                      }
                    }
                  ]
                });
              }
            })();
        }
        return super.createContextMenu(owner, anchor, config);
      }
    })()
  }
});
