# 数字签名

签名功能为实验性功能，后续版本可能发生变更。开发者需谨慎使用。

由于技术原因，web sdk 提供的签名功能需要后端完成部分功能才能正确运行。一个使用 openssl 实现的签名功能后端已随包发布，位于目录 server/pkcs7。

# 后端

# 概述

后端提供了三个接口：计算摘要(http://localhost:777/digest)，对摘要签名(http://localhost:777/sign)，验证签名(http://localhost:777/verify)。

# 启动服务端

进入目录 server/pkcs7，执行npm i安装依赖，其后执行node ./index.js。无意外情况，后端运行于 7777 端口。

# 后端实现

目录 server/pkcs7/bin/](../server/pkcs7/bin)包含了 openssl 可执行程序及[源码，主要功能为计算摘要、签名、验证。开发者可参考实现。

  • 摘要(POST)

    在post请求载体中algorithm参数指定了使用的摘要算法。本示例仅默认提供了一个 sha1 算法。

  • 签名(POST)

    pkcs7 签名算法

  • 验证(POST)

    pkcs7 签名验证。返回值为整形，请参阅 api 手册 Signature_State。

# 前端接口

# 签名

启动 web sdk demo 后，浏览器打开 web sdk，在控制台输入以下 javascript 即可进行签名

var pdfviewer = await pdfui.getPDFViewer();
  //var pdfviewer = new PDFViewerCtrl.PDFViewer({...})
await pdfviewer.registerSignatureHandler(
	//注册签名后端接口，将在签名过程中使用到
	'Adobe.PPKLite',
	'adbe.pkcs7.sha1',
	{
		sign: 'http://localhost:7777/sign', //签名接口
		verify: 'http://localhost:7777/verify', //签名验证接口
		digest: 'http://localhost:7777/digest', //摘要接口
	}
);
var pdfdoc = await pdfui.getCurrentPDFDoc();

file = await pdfdoc.sign({
    rect:{left:0,right:100,top:400,bottom:350},
	pageIndex: 0,
	signer: 'signer',
	reason: 'reason',
	email: 'email',
	DN: 'DN',
	location: 'loc',
	text: 'text',
	filter: 'Adobe.PPKLite',
	subfilter: 'adbe.pkcs7.sha1',
	hashAlgorithm: 0,
});
saveAs(file,"signed.pdf"); //保存到downdloads文件夹

完成签名后，只有下载下来的文档中的签名才符合规则。验证签名建议使用此文档。

# 验证签名

验证签名使用PDFDoc.verifySignature()接口。与签名接口一样，需先注册signatureHandler。

var pdfviewer = await pdfui.getPDFViewer();
  //var pdfviewer = new PDFViewerCtrl.PDFViewer({...})
await pdfviewer.registerSignatureHandler(
	//注册签名后端接口，将在签名及验证过程中使用到，如过签名已注册，此步骤可省略
	'Adobe.PPKLite',
	'adbe.pkcs7.sha1',
	{
		sign: 'http://localhost:7777/sign', //签名接口
		verify: 'http://localhost:7777/verify', //签名验证接口
		digest: 'http://localhost:7777/digest', //摘要接口
	}
);
var pdfdoc = await pdfui.getCurrentPDFDoc();
var pdfform = pdfdoc.getPDFForm();
var signature = pdfform.getField('Signature_0');
if(!signature) throw new Error('signature not found');
var state = await pdfdoc.verifySignature(signature);

获取到的 state 为 signature 的状态，请参阅 api 文档的 Signature_State 变量。
