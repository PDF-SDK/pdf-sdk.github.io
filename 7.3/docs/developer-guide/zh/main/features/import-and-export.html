# Import and Export

# 注释

Annotation数据导入导出支持三种格式：XFDF、FDF 及JSON。下表列出当前SDK中注释支持数据导入导出的情况：

File Type If all annots support What not support
XFDF/FDF 支持大部分 Link
JSON 支持大部分 Screen Image, Link, Sound

# 接口说明

Foxit PDF SDK for Web 当前支持如下接口导入导出注释数据：

Method XFDF/FDF JSON JSON
Import PDFDoc.importAnnotsFromFDF() PDFDoc.importAnnotsFromJSON(annotsJson) PDFPage.AddAnnot(annotJson)
Export PDFDoc.exportAnnotsToFDF() PDFDoc.exportAnnotsToJSON() Annot.exportToJson()

其中接口 PDFPage.AddAnnot(annotJson) 和 Annot.exportToJson() 不支持添加和导出二进制数据。如果您有如下使用场景,建议您根据注释情况进行调整：

  1. 通过接口PDFDoc.exportAnnotsToJSON()导出数据
  2. 通过接口PDFPage.AddAnnot(annotJson)把导出的数据添加回PDF文档

如果您的源文档中含有Stamp 和/或 fileAttachment注释，那么导出的数据会含有二进制数据。 由于目前PDFPage.AddAnnot()接口还不支持二进制数据。以上流程操作就会导致某些注释的AP不显示比如stamp，或附件文件丢失比如fileAttachment。

# 含有二进制数据的注释

  • Stamp
  • fileAttachment

# 表单

表单数据导入导出支持三种标准格式：XFDF、FDF 及XML。

# 接口说明

Foxit PDF SDK for Web 当前支持如下接口导入导出表单数据：

  • PDFDoc.exportFormToFile(fileType)
  • PDFDoc.importFormFromFile(file, isXML)
